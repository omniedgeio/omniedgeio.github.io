Nuestros proyectos: Sistema de monitoreo térmico distribuido de líneas de alta tensión

El sistema de monitoreo térmico distribuido se utiliza en líneas de transmisión de alta tensión para mejorar la fiabilidad y seguridad de las líneas de alimentación. Todo el sistema se combina con sensores alimentados por CT(transformador de corriente), un router alimentado por CT y un sistema SaaS. Utiliza un sistema de protocolo inalámbrico híbrido para comunicarse entre sensores y routers, routers y sistema de control.
